# Human Card Check Pro

Human Card Check Pro is the premium addon for the free Human Card Check plugin.

It adds:

- trust score analysis
- service-backed license validation
- registration risk logs in WordPress
- configurable thresholds for allow or block decisions

## Requirements

- Human Card Check must be installed and active
- a running trust-score service
- a valid Pro token in `Settings > Human Card Check`

## Settings

In `Settings > Human Card Check Pro` you can configure:

- trust service URL
- minimum allow score
- review threshold
- request timeout

## Logs

Recent trust-score decisions are visible in `Settings > Human Card Check Pro Logs`.
